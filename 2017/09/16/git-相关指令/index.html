11 疑问解答

* 指令中的origin 是代表什么意思？
  其实是远程代码仓库的网络地址的替身，每个远程仓库创建时，系统都会默认创建origin,也可以自定义。
  如下：
  新建：git remote add [url point name] [remote url]
  查看： git remote -v。
* git指令后加 --help 可以查询英文帮助文档，如git --help、git init --help，git branch --help等等。
