注意：写博客的Markdown语言参考： Markdown说明 或者 Markdown入门

该md文件在自动生成时会带有一些属性：

  • title: 定义了博文的标题
  • date: 定义了创作博文的时间
  • tags: 定义了博文的标签
    除了这个三个属性以外我们还可以扩展一些属性 ：
  • update: 定义了最后修改的时间
  • comments：定义能否评论此博文(默认为true)
  • categories: 定义了博文的种类

当重装电脑之后，或者想在其他电脑上修改博客，可以使用下列步骤：

1、先安装hexo
$ npm install -g hexo-cli

2、存在github上的git clone下来

git clone git@github.com:LJaer/ljaer.github.io

3、项目文件夹下npm

cd项目名/ npm install –no-bin-links

$ npm install hexo-deployer-git

4、重新配置github和coding的公钥

注意：hexo本地测试运行重启后页面空白,提示 : WARN No layout: index.html，可能是主题文件中的文件为空。
