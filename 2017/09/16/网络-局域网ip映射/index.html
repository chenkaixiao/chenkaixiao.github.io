篇前废话

前阵子由于需要将自己的电脑当做外网服务器，这需要将自己电脑的IP映射到公网中，但是国内很多
相关这方面的工具都是收费的，现在介绍一下国外的 ngrok工具。虽然不是很完美，但是还可以凑合着开发相关的使用。

  • ngrok的特点：
    （1）映射的网络地址会每次重启都会改变。
    （2）服务器在国外，在国内访问相对慢。
